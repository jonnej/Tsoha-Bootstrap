# Tietokantasovelluksen esittelysivu

## Työn aihe

Työn aiheena luoda keskustelufoorumi, jossa käyttäjä rekisteröityä, luoda alueille topiceja, kirjoitella viestejä ja paljon (vähän) muuta.
